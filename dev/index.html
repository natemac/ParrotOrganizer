Screenscraper Credentials

The API requires two separate sets of credentials:
1. Developer credentials — request free API access at screenscraper.fr → Forum → API/Développeurs. You'll receive a devid + devpassword distinct from your login.
2. Member login — your regular site username & password.
Credentials stored locally in dev/scraper-config.json only.
