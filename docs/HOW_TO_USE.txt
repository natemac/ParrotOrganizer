⚠️  IMPORTANT - DO NOT OPEN index.html DIRECTLY!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Opening index.html directly = CORS error = Loading spinner forever

✅ CORRECT:  Use start.bat → Opens http://localhost:8000
❌ WRONG:    Double-click index.html → Opens file:///...

Why? Modern browsers block ES6 modules from file:// protocol.
Solution? Run a local web server (start.bat does this).

🎮 USING THE APP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BROWSING:
• All 454+ games load automatically
• Click any game card for full details
• Scroll through or use filters

SEARCHING:
• Type in search box (top of sidebar)
• Searches game names in real-time

FILTERING:
• Status: All / Installed / Not Installed
• Genre: Shooter, Fighting, Racing, etc.
• Platform: Lindbergh, Namco, Taito, etc.
• Year: Set min/max range
• GPU: Nvidia/AMD/Intel compatibility
• Advanced: Test mode, gun games, 64-bit, etc.

SORTING:
• Dropdown: Name, Year, Platform, Genre, Status
• Arrow button: Toggle ↑ ascending / ↓ descending

VIEW OPTIONS:
• ⊞ Grid view (default) - Cards with icons
• ☰ List view - Compact rows

LAUNCHING GAMES:
• Click "Launch" button on installed game
• Copy the command shown
• Paste into Command Prompt
• Game starts in TeknoParrot!

❌ PROBLEM: Loading spinner forever
✅ SOLUTION: Use start.bat, NOT index.html directly
            Install Python 3 if needed

❌ PROBLEM: "No games found"
✅ SOLUTION: Run scripts\generateGameLists.bat
            Check data\ folder has gameProfiles.txt

❌ PROBLEM: "TeknoParrot installation not found"
✅ SOLUTION: Verify ParrotOrganizer is INSIDE TeknoParrot folder
            Structure should be:
            TeknoParrot\
            ├── TeknoParrotUi.exe
            └── ParrotOrganizer\
                └── start.bat

❌ PROBLEM: Icons not loading
✅ SOLUTION: Check Icons\ folder exists in TeknoParrot
            Access via http://localhost, not file://

❌ PROBLEM: Python not installed
✅ SOLUTION: Download from python.org
            OR use start-node.bat (requires Node.js)
            OR install any web server
